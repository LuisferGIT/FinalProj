Getting and Cleaning Data Course Project

In this repository you will find the final project of the Data Collection and Cleaning Course and it has the description of each of the files contained

• run_analysis.R downloads the dataset and executes the necessary steps according to what is required in the description of the activity.
## Step 1: Merges the training and the test sets to create one data set.
## Step 2: Extracts only the measurements on the mean and standard deviation for each measurement.
## Step 3: Uses descriptive activity names to name the activities in the data set.
## Step 4: Appropriately labels the data set with descriptive variable names.
## Step 5: From the data set in step 4, creates a second, independent tidy data set with the average of each variable for each activity and each subject.

• FinalData.txt is the final file that contains the final data, after applying the steps of the script

• CodeBook.md explains the steps that are carried out to achieve the final result
